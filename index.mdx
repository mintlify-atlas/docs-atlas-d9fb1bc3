Yoko is a WhatsApp chatbot that answers university-related questions. It receives messages through the WhatsApp Cloud API, searches a local JSON knowledge base using fuzzy matching, and generates context-aware answers using the Groq LLaMA3 language model.

## How it works

<Steps>
  <Step title="Message arrives">
    A user sends a WhatsApp message to your registered phone number. Meta's Cloud API forwards it to Yoko's `/webhook` endpoint as a POST request.
  </Step>
  <Step title="Context lookup">
    Yoko's MCP layer uses Python's `difflib` to find the closest matching question in your `data.json` knowledge base (cutoff similarity: 0.4).
  </Step>
  <Step title="LLM response">
    The matched context and the user's original question are sent to the Groq API using the `llama3-8b-8192` model. The response is generated with a temperature of 0.3 for consistent answers.
  </Step>
  <Step title="Reply sent">
    The generated answer is sent back to the user via the WhatsApp Cloud API using `enviar_mensaje`.
  </Step>
</Steps>
